# 股票交易系统 - 前端原型

本文件夹包含股票交易系统的前端页面原型，使用 HTML/CSS 绘制，可直接在浏览器中打开查看。

## 原型文件列表

| 文件名 | 页面名称 | 说明 |
|--------|----------|------|
| [01-登录页.html](01-登录页.html) | 登录页 | 用户登录界面，包含用户名/密码输入框和登录按钮 |
| [02-工具箱首页.html](02-工具箱首页.html) | 工具箱首页 | 系统主入口，展示所有工具卡片 |
| [03-选股工具.html](03-选股工具.html) | 选股工具 | 智能选股功能，支持多种筛选条件 |
| [04-批量回测工具.html](04-批量回测工具.html) | 批量回测工具 | 多股票策略回测分析 |
| [05-个股回测工具.html](05-个股回测工具.html) | 个股回测工具 | 单股票买卖点分析和回测 |
| [06-任务管理.html](06-任务管理.html) | 任务管理 | 定时任务管理和监控 |

## 查看方式

直接在浏览器中打开对应的 HTML 文件即可查看原型效果。

## 设计规范

### 色彩方案
- 主色调: `#1890ff` (蓝色)
- 成功色: `#52c41a` (绿色)
- 危险色: `#ff4d4f` (红色)
- 背景色: `#f0f2f5` (浅灰)
- 文字色: `#262626` (主文字), `#595959` (次要文字)

### 布局规范
- 页面最大宽度: 1400px - 1600px
- 卡片圆角: 8px
- 按钮圆角: 4px
- 间距: 16px, 24px, 32px, 48px

### 字体规范
- 字体族: `-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`
- 标题: 20px - 24px
- 正文: 14px
- 辅助文字: 12px - 13px

## 页面路由结构

```
/login          -> 01-登录页.html
/toolbox        -> 02-工具箱首页.html
/stock-selector -> 03-选股工具.html
/batch-backtest -> 04-批量回测工具.html
/single-backtest -> 05-个股回测工具.html
/task-management -> 06-任务管理.html
```
